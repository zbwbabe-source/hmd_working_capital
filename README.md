# 자금월보 대시보드

Next.js 앱 라우터 + TypeScript로 구현한 자금월보 대시보드입니다.

## 기능

- **Cash Flow**: 현금흐름표(CF) 및 운전자본표 (2025년, 2026년)
- **여신사용현황**: 대리상별 외상매출금, 선수금, 순여신 분석 (2025년)

## 주요 특징

- 트리형 계정과목 구조 (접기/펼치기)
- 스티키 헤더 및 첫 번째 열 고정
- 자동 계산식 적용 (현금흐름, 운전자본)
- 월별 데이터 접기/펼치기 기능
- YoY(전년 대비) 비교 분석
- UTF-8/CP949 인코딩 자동 감지
- 천단위 콤마 표시

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

브라우저에서 http://localhost:3000 접속

## CSV 파일 구조

프로젝트 루트에 다음 폴더 및 파일이 필요합니다:

- `CF/` - 현금흐름표 데이터
  - `2024.csv` - 2024년 데이터 (전년 비교용)
  - `2025.csv` - 2025년 데이터
  - `2026.csv` - 2026년 데이터
- `cashflow/` - 운전자본표 데이터
  - `2024.csv` - 2024년 데이터 (전년 비교용)
  - `2025.csv` - 2025년 데이터
  - `2026.csv` - 2026년 데이터
- `credit/` - 여신사용현황 데이터
  - `2025.csv` - 2025년 12월말 기준

## 기술 스택

- Next.js 14
- TypeScript
